# 👽 Classified Alien Database

> An interactive, retro-OS themed personal website and portfolio simulating a classified database breach.

## 🚀 Overview

Welcome to the **Classified Alien Database — Subject: Owie**. This project is a highly interactive, terminal-inspired personal website built to mimic an extraterrestrial intelligence file system. It features a complete boot-up hacking sequence, a draggable windowed interface, and a custom virtual pet. 

## ✨ Features

* **Interactive Boot Sequence:** Choose between 'Auto-Hack' or manual typing to breach the terminal.
* **Retro OS Interface:** A fully functional desktop environment with a taskbar, start menu, and draggable windows.
* **Virtual Pet (Glerpo):** Feed, wash, and interact with a built-in digital pet complete with a coin economy and mood system.
* **Simulated Security Protocols:** Features immersive "anti-tamper" mechanics (custom clipboard overriding, disabled right-click, and screen locking) to maintain the classified aesthetic.
* **Easter Eggs:** Hidden links, custom pixel cursors, and a layout that updates yearly.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom animations, scanline effects, and retro styling (leveraging `98.css` for classic window aesthetics).
* **Vanilla JavaScript:** DOM manipulation, window dragging logic, clock, and virtual pet state management—no external frameworks required.

## 💻 Local Setup

No build tools or package managers are needed. 
1. Clone the repository or download the source code.
2. Double-click `index.html` to open it in your preferred web browser.

## ⚠️ Disclaimer

*This project includes simulated "malicious" aesthetic behaviors (like locking the screen on certain key presses or overriding the clipboard with redacted text) strictly for immersive storytelling and artistic purposes. It is entirely harmless and contains no actual malware or tracking.*
